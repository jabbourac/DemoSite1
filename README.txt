*** DEMO SITE 1 ***

PURPOSE:
This repository is used for practicing project collaborate and coding skills

SETUP:
1. Accept invitation to collaborate on repository
2. Install git on your local machine
3. In command line, cd into the directory you want to store this project
4. On GitHub, copy the link under "Clone Repository" and run the following command

git clone <URL>

5. Review the resources below formore information on how to use GIT.

RESOURCES:
* GIT Installation - https://www.atlassian.com/git/tutorials/install-git
* GIT Commands Cheatsheet - https://services.github.com/on-demand/downloads/github-git-cheat-sheet.pdf
* Free GIT Course - https://www.udacity.com/course/how-to-use-git-and-github--ud775
* GIT Merging and Branching - https://www.youtube.com/watch?v=GZILYABgAoo
